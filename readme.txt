== Description ==

Плагин Aved Site Scanner - контроль доступности и работоспособности сайтов.
Плагин выполняет проверку сайтов на доступность.
 - Отправляет уведомления в Telegram администратору при неполадоках в работе сайтов.
 - Может запускатся через определенные промежутки времени используя cron.
 - Позволяет в любое время обновить результаты сканирования.
 - Имеет панель настроек для управления списком проверяемых сайтов (добавление, удаление, изменение).
 - Влючает в себя настройки для Telegram-бота.

Более подробную информацию вы найдете на [странице плагина](http://www.).

Если вам понравился мой плагин, то <strong>пожалуйста</strong> поставьте ему 5 звезд.


== Installation ==

1. Скопируйте папку плагина <strong>avss</strong> в <strong>/wp-content/plugins/</strong>. 
2. Активируйте плагин через меню <strong>Плагины</strong>.
3. В главном меню админпанели добавится пункт <strong>Aved Site Scanner</strong>.
4. В подразделе <strong>Sites</strong> добавить в список названия и URL сайтов которые планируеться проверять на доступность.
5. В подразделе <strong>Config</strong> заполнить поля <strong>Bot Token</strong> и <strong>Chat ID</strong>
